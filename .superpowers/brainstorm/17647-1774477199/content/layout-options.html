A: Persistent Sidebar

Sidebar shows project & agent tree. Main area has timeline + event stream. Subagents appear nested under parent in the sidebar. Click to switch context.

Pros

  • Always-visible navigation
  • Natural tree hierarchy
  • Easy multi-agent switching

Cons

  • Less horizontal space for events
  • Sidebar can get cluttered with many agents

B: Top Filters + Agent Tabs

Horizontal layers: project filter > agent tabs (with subagent tabs separated) > event type filters > timeline > full-width event stream. Everything stacks vertically.

Pros

  • Full width for event stream
  • Clear visual hierarchy top-to-bottom
  • Tabs scale well for subagents

Cons

  • Many horizontal bars can stack up
  • Agent list less visible with many agents

C: Icon Rail + Full Main

Narrow icon rail (like VS Code/Slack) with project icons at top, agent avatars below. Click to expand into a panel or switch context. Main area gets maximum space.

Pros

  • Maximum event stream width
  • Familiar pattern (VS Code, Slack)
  • Scales to many projects/agents

Cons

  • Agent names hidden behind icons
  • Extra click to see full agent list
