Behavior

  • Single agent in scope → no agent label (less noise)
  • Multiple agents in scope → agent name appears above event type in a small, semi-transparent font matching the agent's color
  • Subagents prefixed with ↪ to show the parent relationship
  • Border color matches the agent for quick visual scanning
  • Background subtly shifts for subagent rows (#1a1e2e vs #1e1e38) to visually group them
