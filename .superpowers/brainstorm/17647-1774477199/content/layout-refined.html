Collapsed Sidebar View

Icons only: project initials at top, agent circles below. Hover for tooltips. Click the expand handle or any icon to open the full sidebar. Main area gets maximum width.

Key Interactions

  • Select project in sidebar → agent scope bar shows ALL agents in that project
  • Click specific agent(s) in sidebar → scope narrows to just those agents
  • Remove agent chip (×) in scope bar → hides that agent's events
  • Subagent events shown inline with ↪ indicator and distinct border color
  • Collapse handle (‹/›) or drag edge to resize sidebar
  • Timeline bars color-coded per agent, clickable to jump to event
