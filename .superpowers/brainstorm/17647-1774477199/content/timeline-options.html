A: Color Dots Above Icons

Single shared lane. Each event icon gets a small colored dot above it matching the agent's color. Compact, minimal vertical space. Easy to scan which agent produced which event.

Pros

  • Most compact — one line
  • Easy to see interleaved timing
  • Dot color links to sidebar & stream

Cons

  • Can get crowded with many agents
  • Harder to follow one agent's flow

B: Separate Lanes Per Agent

Each agent (and subagent) gets its own horizontal lane. Subagent lanes are slightly indented and smaller. Clear per-agent activity at a glance. Lanes auto-appear/disappear based on scope.

Pros

  • Crystal clear per-agent activity
  • Easy to see timing relationships
  • Subagent nesting is obvious

Cons

  • Takes more vertical space
  • Can grow tall with many agents

Both options include:

  • Click any icon → auto-scrolls event stream to that event (highlighted briefly)
  • Hover → tooltip with event type + summary
  • Scrolling animation → events slide left as time progresses (existing behavior)
  • Time range buttons → 1m / 5m / 10m zoom levels
  • Adapts to scope → single agent = option A behavior automatically
